PIRATE WARS
"Networked multiplayer combat using Photon Fusion 2's Host-Client architecture."


Pirate Wars is a multiplayer FPS where players battle on moving pirate ships. The game uses server-authoritative networking to ensure fair gameplay, with the host validating all damage, scoring, and win conditions.

Developed in Unity using C#, Photon Fusion 2, Photon Advanced KCC and Photon Finite State Machine.
