GUNS BELOW
"Designed and programmed a modular gameplay framework."


Guns Below is a 3D combat prototype built around a component-based gameplay framework. The system features modular weapon classes, centralized management using singletons, and AI-driven enemy behavior designed to react dynamically to player movement and actions.

The game’s architecture emphasizes scalability and flexibility — weapons, effects, and interactions are all designed to be reusable components that can easily integrate into other Unity projects.

Developed in Unity and implemented using C#.
